## New Features

### Migration System
- `migrate_data(admin, source_contract, expected_version)` - restores instance storage from old contract
- Getter functions: `get_admin`, `get_protocol_version`, `get_total_supply_raw`, `get_next_claim_id`, `get_next_proposal_id`, `get_migration_status`
- `MigrationStatus` and `MigrationResult` types
- `MigrationCompleted` event

### Frontend
- `/about` page now loads contract IDs dynamically from `@kchng/shared`
- Single source of truth: `packages/shared/src/networks.ts`
